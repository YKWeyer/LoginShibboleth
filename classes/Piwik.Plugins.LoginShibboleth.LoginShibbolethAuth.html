\Piwik\Plugins\LoginShibboleth

\Piwik\Plugins\LoginShibbolethLoginShibbolethAuth

LoginShibbolethAuth does the authentication.

This is the overridden Auth class of native Login Plug-in in Piwik. It handles all the login request and API queries to the plug-in. The class only changes the normal login and everything else is the same so the API functions still could work. Any authentication related settings should be done here.
