\Piwik\Plugins\LoginShibboleth

\Piwik\Plugins\LoginShibbolethShibbolethAdapter

ShibbolethAdapter is the Shibboleth data retrieval adapter.

LoginShibboleth has two ways of retrieval of user data and access. The first is the ShibbolethAdapter which directly interpret the data from Shibboleth and return the user information. The other one is LdapAdapter. Any changes to the way shibboleth acts can be reflected here.
