\Piwik\Plugins\LoginShibbolethLoginShibboleth

Main Login Shibboleth Settings.

Here the different JavaScript, function and activation hooks are added. If there is another JavaScript file is to be added or functions needed to be run before or after this plug-in activation use this class to reflect them.
