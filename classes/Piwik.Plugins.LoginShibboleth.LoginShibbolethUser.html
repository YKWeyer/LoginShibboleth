\Piwik\Plugins\LoginShibbolethLoginShibbolethUser

LoginShibbolethUser is the user Model class for Shibboleth.

This class inherit most of the functions from the Model class of Piwik UsersManager. Any changes to default functions of Model should be added here. The overridden functions of Model are getUser and __construct.
